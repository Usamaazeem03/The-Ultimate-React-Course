What is React?
React is extermely popular, declarative, component based, state deiven Javascript library for building user interfaces, created by facebook. // Thanks facebook

_Based on components:
Components are the building blocks of user interfaces in React.

_Deciarative:
We describe how components look like and hw they work using a declarative syntax called JSX.

_State driven:
React reacts to state changes by re-rendering the UI.

_JavaScript library:
Because react is only the "view" layer. we need to pick multiple external libraries to build a complete application.

_Extemely popular:
Accordin to the weekly download numbers form npm Reat is most used framework.

_Created by facebook:
React was created in 2011 (open-source in 2013) by Jodan Walke an engineer working at facebook at the time.
